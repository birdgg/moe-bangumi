# MoeBangumi
MoeBangumi 是一款追番工具
Core Stack: Rust, React, Claude Code with Opus 4.5

## 目标
- 订阅新番
- 收藏 bd
- 迁移 B 站收藏的番剧
- 简化用户操作
- Avoid $ success at all cost
